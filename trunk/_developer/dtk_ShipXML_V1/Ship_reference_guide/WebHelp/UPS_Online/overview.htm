OnLine Tools Overview

This Reference Guide is designed for UPS customers to use during the integration of their systems with UPS OnLine Tools. Although the OnLine Tools can be implemented using a number of different programming languages, this Reference Guide focuses on the details related to Java and Visual Basic.

 

UPS OnLine Tools are server applications based on Hypertext Transport Protocol (HTTP) and Extensible Markup Language (XML). The OnLine Tools provide an easy method for integrating UPS functionality into your company's e-commerce applications.

 

Building your OnLine Tools application requires:

 

  1. Communication with the UPS OnLine Tools server.

  2. The submission of a request message.

  3. The parsing of a response message.

  4. Error Handling.

How it works:  

 

 

This Reference Guide supports development of UPS OnLine Tools using either Java or Visual Basic.
